# World ID Verification (Vite + React)

This project demonstrates **human verification using Worldcoin’s IDKit** in a **Vite + React** frontend, with **proof verification handled on a backend server**.

---

## Features

- ✅ Worldcoin **World ID (IDKit)** integration
- ✅ Proof verification sent to backend
- ✅ Supports **Device verification level**

---

## Tech Stack

- **Frontend:** Vite + React
- **World ID SDK:** @worldcoin/idkit
- **Backend:** Express / / NestJS (any)
- **Verification:** Worldcoin Proof API
